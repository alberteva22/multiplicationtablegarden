A Szorzótábla kert egy saját ötlet alapján, de ChatGPT segítségével készített, vibe- kódolású, szorzótáblát gyakoroltató játék, ahol egyszerre kettő vagy 4 gyerek játszhat egy digitális érintőkijelzős tábla előtt, de nem egymás legyőzése a céál, hanem a kooperáció. Az "edzés" gombnál végig lehet gyakorolni a szorzótáblát sorban, míg a "Mix" gombnál egy adott tényező esetében csak 3 szorzatot kell kiszámolni. A csúszkán beállítható a szorzandó.
/The Multiplication Table Garden is a vibe-coded multiplication table practice game, based on my own idea but created with the help of ChatGPT, where two or four children can play at the same time in front of a digital touchscreen board, but the goal is not to defeat each other, but to cooperate. The "training" button allows you to practice the multiplication table in sequence, while the "Mix" button allows you to calculate only three products for a given factor. The multiplier can be set using the slider.
